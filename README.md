# funscript workspace

It's a funscript player and editor that works with video and audio.

## Running this software

### Using the docker image

```sh
docker run --rm \
  -p 80/tcp \
  --name fsworkspace \
  tin9tt/funscript-workspace:latest
```

### From src

Download the most suitable src from the repository.

Then:

```sh
bun install
bun build
bun start
```
